React 的核心原理主要围绕以下几个方面：

1. 虚拟DOM (Virtual DOM)

工作原理：

  • React 创建虚拟DOM树（轻量级的JS对象）
  • 状态变化时，生成新的虚拟DOM树
  • 通过Diff算法比较新旧树的差异
  • 批量更新实际DOM（最小化操作）

Diff规则：

  1. 同级比较：只比较同一层的节点
  2. Key的重要性：帮助React识别元素的移动、添加、删除
  3. 类型不同则重建：元素类型改变，整个子树重建

3. Fiber架构（React 16+）

Fiber核心概念：

  • 增量渲染：将渲染工作拆分成多个小任务
  • 可中断：React可以暂停、恢复渲染
  • 优先级调度：为不同类型的更新分配优先级
